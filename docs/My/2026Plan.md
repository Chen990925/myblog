很难想象，当时在刚在使用GPT3.5时 那种看着对方逐字逐句对我的任何问题都能回答的兴奋感，想不到现在26年了，claude code ，openai codex，gemini 这些模型已经进化到了如此地步。。
包括agent，openclaw这种除了chat模式能真正帮你执行任务的ai助手，感觉再过两年可能ai要成精了。。

工作快四年了，其实工作也是大部分仅限于CRUD，之前为了找工作还能努力学习新知识（虽然都没用上），工作之后强度大出于私心就懈怠了。为了不变成老古董，2026年务必要借助ai完成个人积累的蜕变，以前哪些看着

  1.git 熟练使用，不仅限于拉取提交推送，之前一直好奇各个ide的各类git操作是什么，又想知道又觉得没用就得过且过。目标:整理成完整知识体系，对于每个ide的git界面和出现的问题都能熟练解决
  2.ai agent开发方向了解学习，不知道不可怕，先从第一步做起
  3.部署 openclaw 且能接入主流平台
  4.熟练使用docker相关技术
  5.上线自己的网站或项目
    ideal:github使用教学指南；搭建一个查询github上最近一周/月/季度最高stars积累数量/速度的中文查询网站，能对英文网站进行快速翻译，快速介绍项目结构，可能涉及ai技术
    微信小程序:类似flomo的备忘录小程序，使用微信云原生技术
